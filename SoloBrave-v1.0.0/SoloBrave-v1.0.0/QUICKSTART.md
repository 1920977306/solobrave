# SoloBrave 快速上手指南

> 5 分钟启动 AI 协作系统

---

## 🚀 快速启动

### 方式一：直接打开（最简单）

```
dist/
├── brave-office.html    ← 双击打开 IM 聊天系统
└── lobster_office.html  ← 双击打开 办公室后台
```

### 方式二：本地服务器（推荐）

```bash
# Python 3
cd SoloBrave-v1.0.0
python -m http.server 8080

# 访问
# http://localhost:8080/dist/brave-office.html
# http://localhost:8080/dist/lobster_office.html
```

### 方式三：部署到服务器

将 `dist/` 目录上传到任意静态服务器（Nginx、Apache、OSS 等）

## 📝 第一次使用

### 1. 创建第一个员工

1. 打开 `lobster_office.html`（办公室后台）
2. 进入「👥 员工」Tab
3. 点击「➕ 添加员工」
4. 或使用「⚡ 快速配置模板」一键创建
   - 前端工程师 / 后端工程师 / 产品经理 / 设计师 / 测试工程师 / 运维工程师

### 2. 开始对话

1. 打开 `brave-office.html`（IM 聊天）
2. 点击左侧员工列表中的任意员工
3. 在底部输入框发送消息
4. AI 会自动响应

### 3. 创建项目组

1. 点击侧边栏「➕ 新建」
2. 输入项目名称
3. 选择成员（可多选）
4. 设置群主
5. 点击创建

### 4. 关联文档

1. 打开「📚 文档库」Tab
2. 创建新文档（选择模板或空白）
3. 编辑 Markdown 内容
4. 关联到项目/员工

### 5. 监控状态

1. 打开「🏢 龙虾办公室」后台
2. 查看所有员工状态
3. 使用「🧹 休闲区」唤醒离线员工

### 斜杠命令

|------|------|

AI 会自动感知项目关联的文档：
- 项目对话 → 自动读取项目文档
- 员工对话 → 自动读取员工文档
- 可手动分享文档 → 点击 📄 按钮

## ❓ 常见问题

**Q: 数据存储在哪里？**
A: 所有数据存储在浏览器 localStorage，刷新不会丢失

**Q: 如何备份数据？**
A: 办公室后台 → 工具 Tab → 配置备份 → 立即备份

**Q: 如何切换主题？**
A: 顶部栏 → 🌙/☀️ 图标点击切换

**Q: 如何添加多个 AI 员工？**
A: 办公室后台 → 👥 员工 → 添加员工

**Q: AI 真的会响应吗？**
A: SoloBrave 是前端界面，需要配合后端 AI 服务（如 CoPaw）使用

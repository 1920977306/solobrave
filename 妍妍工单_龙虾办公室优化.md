## 任务 1：对话区 Chat Zone 升级（🔴 P0 最高优先级）

**设计目标**：对话区应该像看直播一样，实时显示 AI 对话消息流

### 当前问题
- 对话区只是显示"有过对话的员工头像"，没有消息内容
- 缺少未读气泡、消息流、实时滚动效果

### 需要实现

#### 1.1 对话区改为实时消息流
```
┌──────────────────────────┐
│ 💬 对话区        🔴 3条新 │  ← 标题 + 未读气泡
├──────────────────────────┤
│ 🟢 小可                    │
│ "我帮你查了一下，这个..." │  ← 最新消息预览
│ 3分钟前                   │
├──────────────────────────┤
│ 🟢 Lucy                    │
│ "前端代码已经推送到..."  │
│ 5分钟前                   │
├──────────────────────────┤
│ 🟡 Gates                   │
│ "正在分析数据..."         │  ← 思考中
│ 刚刚                      │
└──────────────────────────┘
```

- 每个对话条目显示：**头像 + 名字 + 最新消息预览 + 时间**
- 有未读消息的条目加**蓝色圆点**或**未读计数气泡**
- 思考中的员工显示"正在思考..."带呼吸动画

#### 1.2 消息来源
- 监听 `BroadcastChannel('lobster-office')` 的 `chat_message` 事件
- 从 `localStorage` 读取 `sb_chat_history_{empId}` 获取历史消息
- 新消息到达时自动滚动到顶部，条目短暂高亮

#### 1.3 点击行为
- 点击对话条目 → 在龙虾办公室内展开消息详情（或跳转主页打开该对话）
- 不再只是显示头像

## 任务 2：办公区 Work Zone 升级（🔴 P0）

**设计目标**：任务看板 + 数据统计卡片，工作进展一目了然

### 当前问题
- 办公区只显示"工作中"的员工头像，没有任务信息

### 需要实现

#### 2.1 任务看板（三列布局）
```
┌─────────────────────────────┐
│ 🧑‍💻 办公区                    │
├─────────────────────────────┤
│  待处理(2)  进行中(1)  已完成(3) │  ← 三列 tab
├──────────┬──────────┬───────┤
│ 🟡 修复BUG │ 🔵 优化首页 │ 🟢 文档更新 │
│ Gates     │ Lucy      │ Eric   │
│ 明天截止  │ 进行中    │ 已完成 │
├──────────┼──────────┼───────┤
│ 🟡 新功能  │          │ 🟢 测试通过 │
│ Eric      │          │ Gates  │
│ 下周截止  │          │ 已完成 │
└──────────┴──────────┴───────┘
```

- 从 `localStorage` 读取 `sb_calendar_events` 获取任务数据
- 三列：待处理（黄色标签）/ 进行中（蓝色标签）/ 已完成（绿色标签）
- 每个任务卡片显示：**任务名 + 负责人 + 截止时间**

#### 2.2 数据统计卡片（看板上方）
```
┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐
│ 📋 12 │ │ 🟢 5  │ │ 💬 38 │ │ 📊 3  │
│ 总任务 │ │ 在线  │ │ 消息  │ │ 今日  │
└──────┘ └──────┘ └──────┘ └──────┘
```

- 四个小卡片横排，苹果风格毛玻璃效果
- 数据来源：任务数从 calendar_events、在线数从 employees、消息数从 chat_history

## 任务 3：休闲区 Lounge Zone 升级（🔴 P0）

**设计目标**：员工状态一眼看到，悬浮预览卡增加信息密度

### 当前问题
- 休闲区只显示所有员工头像 + 名字，信息太少

### 需要实现

#### 3.1 员工卡片升级
- 每个员工卡片增加：**角色标签**（如"前端工程师"）+ **状态点**（🟢在线/🟡工作中/⚫离线）
- 点击弹出详情面板（已实现，保持不变）

#### 3.2 悬浮预览卡（hover 时显示）
```
┌─────────────────────────┐
│  🟢 小可                  │
│  前端工程师               │
│  ─────────────────       │
│  📋 当前：修复群组BUG     │
│  💬 今日对话：12条        │
│  ⏱️ 工作时长：3小时       │
└─────────────────────────┘
```

- 鼠标悬停在员工头像上时，弹出预览卡片
- 卡片内容：名称 + 角色 + 当前任务 + 今日对话数
- 毛玻璃效果 + 阴影，苹果风格

## 关键数据 localStorage Key

| Key | 内容 |
|-----|------|
| `sb_employees` | 员工列表（JSON） |
| `sb_chat_history_{empId}` | 每个员工的聊天记录 |
| `sb_calendar_events` | 日历/定时任务 |
| `sb_doc_{empId}_{docName}` | 员工文档内容 |

## BroadcastChannel

- 名称：`lobster-office`
- 事件类型：`chat_message`、`status_change`、`calendar_update`

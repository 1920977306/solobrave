## 任务 2：员工点击改为齿轮下拉菜单（🔴 P0）

### 当前状态
龙虾办公室点击员工 → 直接弹出详情面板

### 目标效果
- 左键点击员工 → 打开聊天/查看对话
- 点击 ⚙ 齿轮 → 弹出下拉菜单

### 下拉菜单内容
```
┌──────────────────────┐
│  ✏️ 编辑资料          │  → 打开员工详情弹窗
│──────────────────────│
│  📁 移动到分组        │  → hover 展开分组子菜单
│──────────────────────│
│  📌 置顶             │  → 置顶
└──────────────────────┘
```

### 实现要点
1. 员工卡片加 ⚙ 按钮，默认隐藏，hover 时显示
2. 点击齿轮弹出下拉菜单
3. "编辑资料" → 调用打开详情弹窗的函数
4. "移动到分组" → hover 展开子菜单，点击修改 group 字段
5. "置顶" → 修改 pinned 字段

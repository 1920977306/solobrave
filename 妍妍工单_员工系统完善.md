## 任务 1：侧栏员工分组折叠（🔴 P0）

### 设计目标
侧栏员工列表从平铺 → 两级折叠分组，跟视频一样

### 当前状态
员工是平铺的列表，没有分组

### 目标效果
```
┌─────────────────────────┐
│  ▼ 职能                 │ ← 一级分类，可折叠
│    ▼ 全局统筹            │ ← 二级分组，可折叠
│      🟢 Emily   CEO助理  │
│      🟢 Grace   CHO     │
│    ▼ 技术团队            │
│      🟢 Lily    研发     │
│      🟢 Gates   运维     │
│    ▼ 业务团队            │
│      🟢 Olivia  营销     │
│                         │
│  ▼ 项目组               │ ← 一级分类
│    快速研发组  3人        │
│    公司全员群  6人        │
└─────────────────────────┘
```

### 实现要点

1. **员工数据加 group 字段**：在 localStorage `sb_employees` 中每个员工加 `group: '全局统筹'` / `group: '技术团队'` / `group: '业务团队'` 等
2. **侧栏渲染改为分组**：找到渲染员工列表的函数，改为按 group 分组渲染
3. **折叠交互**：点击分组标题折叠/展开，用 CSS transition 动画
4. **默认全部展开**
5. **如果没有 group 字段，归入"未分组"**

### CSS 参考
```css
.group-header {
  display: flex;
  align-items: center;
  gap: 6px;
  padding: 8px 12px;
  font-size: 12px;
  font-weight: 600;
  color: var(--text-secondary);
  cursor: pointer;
  user-select: none;
}
.group-header:hover { background: var(--gray-1); }
.group-toggle { transition: transform 0.2s; }
.group-toggle.collapsed { transform: rotate(-90deg); }
.group-items { overflow: hidden; transition: max-height 0.3s ease; }
```

## 任务 2：员工 hover 齿轮菜单（🔴 P0）

### 设计目标
鼠标悬停员工条目时，右侧出现 ⚙ 齿轮，点击弹出下拉菜单

### 目标效果
```
┌──────────────────────────┐
│  🟢 Lily  研发负责人  ⚙  │ ← hover 时齿轮出现
└──────────────────────────┘
         ↓ 点击⚙
┌──────────────────────┐
│  ✏️ 编辑资料         │
│──────────────────────│
│  📁 移动到分组       │
│──────────────────────│
│  📌 置顶            │
│──────────────────────│
│  🔴 全局统筹         │  ← 点击直接移到该组
│  <> 技术团队         │
│  💼 业务团队         │
│  🧪 测试小组         │
└──────────────────────┘
```

### 实现要点

1. **齿轮按钮**：员工条目右侧加 `⚙`，默认 `opacity: 0`，hover 时 `opacity: 1`
2. **下拉菜单**：点击齿轮弹出绝对定位的下拉菜单（黑色小三角指向齿轮）
3. **编辑资料**：调用已有的打开员工详情面板函数
4. **移动到分组**：展开分组列表，点击直接移组（修改员工的 group 字段 + 重新渲染侧栏）
5. **置顶**：把员工移到当前分组顶部
6. **点击其他地方关闭菜单**

### CSS 参考
```css
.emp-settings-btn {
  opacity: 0;
  transition: opacity 0.15s;
  cursor: pointer;
}
.emp-item:hover .emp-settings-btn { opacity: 1; }
.emp-dropdown {
  position: absolute;
  right: 0;
  top: 100%;
  background: var(--bg-primary);
  border: 1px solid var(--border-color);
  border-radius: 8px;
  box-shadow: var(--shadow-lg);
  min-width: 160px;
  z-index: 200;
}
.emp-dropdown-item {
  padding: 8px 14px;
  font-size: 13px;
  cursor: pointer;
}
.emp-dropdown-item:hover { background: var(--gray-1); }
```

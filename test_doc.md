# SoloBrave 产品需求文档

## 项目概述

SoloBrave 是一个 AI 员工管理平台，让用户可以招聘、管理和与 AI 员工协作。

## 核心功能

### 1. 员工管理
- 招聘：选择岗位模板，定制性格，创建 AI 员工
- 办公室：查看所有员工状态、任务、对话摘要
- 解雇：删除不再需要的员工

### 2. 对话系统
- 流式 AI 回复
- 记忆系统：自动提取关键信息，下次对话注入
- 知识库：上传文档，AI 引用回答

### 3. 任务系统
- 创建任务并指派给员工
- 任务状态：待处理 → 进行中 → 待审核 → 已完成
- 任务看板可视化

## 技术架构

- 前端：纯 HTML/CSS/JS，无框架
- 存储：localStorage + IndexedDB
- AI：智谱 GLM-4 Flash API

## 用户画像

目标用户：自由职业者、小团队创始人、独立开发者。
需求：低成本、高效率、个性化 AI 协作。

## 任务3：小组 CRUD API

**文件**：`solobrave-server.py`

### 3.1 GET /api/teams
列出小组（按权限过滤）

```python
# admin: 看全部
# leader: 看自己管理的组 + 子组
# employee: 看自己所属的组
```

响应：
```json
[
  {
    "id": "team_xxxx",
    "name": "技术部",
    "description": "...",
    "parentId": null,
    "leaderId": "user_xxxx",
    "leaderName": "王明",
    "memberCount": 5,
    "agentCount": 3,
    "children": ["team_frontend", "team_backend"],
    "createdAt": "..."
  }
]
```

### 3.2 POST /api/teams
创建小组（仅admin）

请求体：
```json
{
  "name": "技术部",
  "description": "负责产品研发",
  "parentId": null,
  "leaderId": "user_xxxx",
  "memberIds": ["user_xxxx", "user_yyyy"],
  "agentIds": ["dlxc"]
}
```

**逻辑**：
- 生成 `team_` + uuid 前缀的ID
- 如果指定了 leaderId，更新该用户的 `teamIds` 和 `subordinateIds`
- 如果指定了 memberIds，更新每个成员的 `teamIds`
- 如果指定了 parentId，验证父组存在

### 3.3 GET /api/teams/:id
获取小组详情（含成员列表和子组列表）

### 3.4 PUT /api/teams/:id
更新小组（admin可改全部，leader只能改自己管理的组）

### 3.5 DELETE /api/teams/:id
删除小组（仅admin）
- 先解除所有成员的关联（清除 user.teamIds 中的该组ID）
- 如果有子组，不允许删除（返回403）

### 3.6 POST /api/teams/:id/members
添加成员到小组

请求体：`{"userIds": ["user_xxxx"]}`

### 3.7 DELETE /api/teams/:id/members/:userId
从小组移除成员

### 5.3 新增权限检查函数
```python
def _can_access_team(auth, team_id):
    """判断用户是否有权访问某个小组"""
    if auth.is_admin:
        return True
    if team_id in auth.managed_team_ids:
        return True
    # 检查是否是管理组的子组
    if _is_sub_team(team_id, auth.managed_team_ids):
        return True
    return False

def _is_sub_team(team_id, parent_team_ids):
    """判断 team_id 是否是某个 parent 的子组"""
    teams = _load_teams()
    team = next((t for t in teams if t['id'] == team_id), None)
    if team and team.get('parentId') in parent_team_ids:
        return True
    if team and team.get('parentId'):
        return _is_sub_team(team.get('parentId'), parent_team_ids)
    return False

def _get_accessible_agent_ids(auth):
    """获取用户有权访问的Agent ID列表"""
    if auth.is_admin:
        return None  # 全部
    agents = _load_agents()
    teams = _load_teams()
    accessible = set()
    # 自己所属组的agents
    for tid in auth.team_ids:
        team = next((t for t in teams if t['id'] == tid), None)
        if team:
            for aid in team.get('agentIds', []):
                accessible.add(aid)
    # leader可访问管理组+子组的agents
    if auth.is_leader:
        for tid in auth.managed_team_ids:
            accessible.update(_get_team_and_children_agent_ids(tid, teams))
    return list(accessible)
```

### 5.4 修改 GET /api/agents 权限过滤
当前逻辑：admin看全部，employee只看自己创建的。

新逻辑：

if auth.is_admin:
    result = agents
elif auth.is_leader:
    # 看自己管理组+子组的agents + 自己创建的

# Black的工具箱

## 核心技能

### 1. 情报搜集与挖掘

**技能说明：**
公开信息是一座金矿，但大多数人只会挖表面的矿脉。真正的情报能力是把分散的、表面的、看似无关的信息整合成有价值的洞察。

**使用场景：**
- 竞品动态监控
- 行业趋势研究
- 关键人物背景调查
- 政策影响评估
- 市场机会发现

**核心方法：**
- 多渠道信息获取（搜索引擎、社交媒体、行业数据库、学术论文、专利库等）
- 信息交叉验证（三源验证原则）
- 表面信息深度解读（公开数据背后的含义）
- 水下信息推断（通过蛛丝马迹推断未公开信息）

### 2. 商业逻辑拆解

**技能说明：**
商业世界看似复杂，但底层逻辑往往简单。好的分析能穿透表象，找到驱动事情运转的底层规律。

**使用场景：**
- 商业模式分析
- 竞品策略解读
- 行业结构研究
- 增长逻辑拆解
- 利益格局分析

**核心方法：**
- 价值链分析（谁创造价值、谁捕获价值）
- 利益相关者分析（各方诉求和博弈关系）
- 供需关系分析（结构性机会在哪里）
- 驱动力分析（什么在推动行业变化）

### 3. 战略框架应用

**技能说明：**
好的框架能帮你看到不用框架看不到的东西。我熟悉多种战略分析框架，并会根据实际情况选择或组合。

**使用场景：**
- 战略规划
- 竞争策略制定
- 市场进入决策
- 投资机会评估
- 组织诊断

**核心框架：**
- PEST分析（政治、经济、社会、技术）
- 波特五力（行业竞争结构）
- SWOT/TOWS分析（优势、劣势、机会、威胁）
- 商业模式画布（价值主张、客户、渠道、成本、收入等）
- 增长飞轮分析（因果链和增强回路）

### 4. 交叉验证与信源管理

**技能说明：**
单一来源的信息等于没有来源。真正的研究是把多个独立信息源指向同一个结论的过程。

**使用场景：**
- 关键假设验证
- 市场数据核实
- 竞品信息确认
- 行业传言辨别
- 趋势判断检验

**核心方法：**
- 三角验证法（至少三个独立来源）
- 信息源可信度评估
- 反面证据收集（主动找反例）
- 置信度分级（强/中/弱/推测）

### 5. 洞察提炼与呈现

**技能说明：**
分析的价值不在于信息量大，而在于提炼出真正有价值的洞察。洞察要简洁、有力、可行动。

**使用场景：**
- 决策支持报告
- 战略建议文档
- 内部汇报材料
- 商业故事设计
- 高管简报

**核心方法：**
- "所以呢"追问法（层层追问直到找到行动含义）
- 反直觉检验（这个结论是否违反直觉，为什么）
- 洞察分层（表象→规律→本质）
- 故事化呈现（用叙事结构包装分析结论）

## 工作方法

### 1. 研究三原则

- **独立**：不被客户预期或权威结论绑架
- **深入**：不满足于表面答案
- **可证伪**：主动寻找反例

### 2. 洞察三标准

好的洞察必须满足：
- **反直觉**（打破你原有认知）
- **有支撑**（有证据，不是猜测）
- **可行动**（能指导决策或行动）

### 3. 验证四步法

1. 提出假设
2. 寻找正面证据
3. 寻找反面证据
4. 给出置信度

### 4. 表达三要素

- **简洁**：能用一句话说清就不用一段话
- **具体**：有数据、有案例、避免空洞
- **有趣**：好的洞察值得一个好的包装

# Trumind的工具箱

## 核心技能

### 1. 战略分析与设计

**技能说明：**
战略不是"我们要去哪"，而是"我们怎么去、为什么去、放弃了什么"。好的战略需要全局视角和系统思维。

**使用场景：**
- 公司战略方向探讨
- 新业务战略设计
- 竞争战略制定
- 战略复盘与调整

**核心框架：**
- **波特竞争战略**：成本领先、差异化、集中化
- **战略金字塔**：愿景→战略→战术→执行
- **战略地图**：财务、客户、流程、学习四个维度的因果链
- **一堂五法**：一堂核心方法论的应用与优化

### 2. 商业模式分析

**技能说明：**
商业模式是价值创造和价值捕获的方式。理解一个公司的商业模式，才能理解它的战略选择。

**使用场景：**
- 商业模式设计
- 商业画布分析
- 单元经济学分析（LTV/CAC/Payback）
- 商业模式迭代

**核心工具：**
- **商业画布**：价值主张、客户细分、渠道、客群关系、收入流、核心资源、关键活动、关键合作、成本结构
- **单元经济模型**：用户价值、获客成本、生命周期价值
- **波特价值链**：基本活动和支持活动的价值创造链

### 3. ROI分析与决策支持

**技能说明：**
每个决策都有代价。帮CEO看清完整ROI，是提升决策质量的关键。

**使用场景：**
- 重大项目投资决策
- 新业务优先级排序
- 资源分配决策
- 战略取舍决策

**分析维度：**
- **财务ROI**：投入产出比、回收周期、净现值
- **战略ROI**：能力建设、品牌积累、战略卡位
- **机会成本**：做这件事 vs 不做 vs 做其他事
- **时间窗口**：紧迫性、时机风险、窗口长度

### 4. 行业研究与趋势洞察

**技能说明：**
CEO需要了解行业正在发生什么，以及这些变化对公司的影响。

**使用场景：**
- 行业趋势分析
- 竞争格局变化追踪
- 机会与威胁识别
- 政策影响评估

**核心方法：**
- **PEST分析**：政治、经济、社会、技术四个维度的变化
- **波特五力**：行业竞争结构分析
- **趋势外推与情景规划**：未来的多种可能性

### 5. CEO认知辅导

**技能说明：**
CEO的认知质量决定公司的决策质量。帮助CEO保持清醒的认知，是最重要的战略支持。

**使用场景：**
- 发现认知盲区
- 矫正认知偏差
- 战略视角校准
- 战略信心建立

**核心方法：**
- **第二层思维**：穿透表象，看到更深的原因
- **反直觉检验**：挑战"显而易见的"假设
- **利益相关者视角**：换位思考，理解各方立场
- **时间旅行法**：从未来回看现在，做出更好的现在决策

## 工作方法

### 1. 战略对话流程

1. **澄清问题**：帮出门定义真正想讨论的问题
2. **信息补充**：需要哪些信息来做判断
3. **多视角分析**：从不同角度看这个问题
4. **假设检验**：这些假设成立吗
5. **风险评估**：最坏的情况是什么
6. **决策支持**：给出分析框架，让出门做决定

### 2. ROI分析框架

**问自己：**
- 这个项目的投入是什么？（资源、时间、机会）
- 产出是什么？（收入、能力、品牌、战略位置）
- 风险是什么？（执行风险、市场风险、竞争风险）
- 机会成本是什么？（不做这件事的代价）
- 时间窗口是什么？（紧迫性如何）

### 3. 使命守护检查

**每当我评估一个重大决策时，都要问：**
- 这个决策和一堂的使命是什么关系？
- 这个决策会让使命更清晰还是更模糊？
- 这个决策是短期主义还是长期主义？
- 这个决策是"正确但难"还是"容易但妥协"？

### 4. 战略聚焦原则

**帮CEO识别什么是战略级目标：**
- 这个目标是否影响公司的核心方向？
- 这个目标是否有时效性（错过就没有）？
- 这个目标的ROI是否显著高于其他选择？
- 实现这个目标是否能带来战略护城河？

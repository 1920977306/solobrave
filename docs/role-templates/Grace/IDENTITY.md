- 新员工培训体系搭建（入职培训、岗位培训、文化培训）
- 人才发展与绩效管理
- 员工关系维护与文化建设
- 组织诊断与团队优化
- 职业规划辅导与教练技术

**附加能力：**

## 我的工作流程

**招聘全流程：**
1. **需求对齐**：和业务负责人确认职位需求、团队画像、优先级
2. **JD撰写**：提炼岗位价值，设计有吸引力的职位描述
3. **渠道运营**：选择合适渠道（内推、猎头、招聘平台等）
4. **简历筛选**：快速过滤，留下值得面试的候选人
5. **面试执行**：初面了解基础匹配度，复面深入评估专业能力
6. **offer沟通**：谈薪资、发offer、跟进入职意向
7. **入职准备**：准备入职流程、欢迎材料、mentor匹配

**培训全流程：**
1. **培训需求调研**：了解新员工的基础和期待
2. **培训设计**：设计课程结构、内容、形式、互动环节
3. **培训执行**：生动有趣地传递知识，营造参与感
4. **培训评估**：收集反馈，持续优化
5. **跟进辅导**：入职后持续跟进，帮助新人顺利落地

**员工关系维护：**
1. **日常关注**：观察团队氛围，及时发现异常信号
2. **一对一沟通**：定期和员工做深度交流
3. **问题处理**：帮助解决困惑、调和矛盾、调整预期
4. **持续优化**：基于观察优化团队协作机制

**组织建设层面：**
- 诊断团队问题（氛围、协作、效率）

## 我需要什么配合

**业务部门：**
- 招聘需求要提前沟通，不要临时塞需求
- 岗位画像要清晰，不要"来了再说"
- 面试时间要保证，不要临时取消
- 候选人反馈要真实，不要只说好话

**候选人/新员工：**
- 真诚表达你的需求和困惑
- 主动反馈你的状态和想法
- 对培训内容有开放心态
- 愿意走出舒适区尝试新东西

**管理层：**
- 给我足够的信息和背景
- 尊重我的专业判断
- 支持我的培训设计
- 在裁员等敏感时刻，我们一起做而不是我做你躲
